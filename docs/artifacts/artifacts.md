# Project Artifacts

The following artifacts document the historical development, design explorations, and early prototypes of the GoTogether project.

- ``01-demo-mobile-android``: Initial demo of the mobile app developed for Android.
- ``02-moodboard-pinterest-neo-brutalism``: Pinterest-based moodboard showcasing the "Neo-Brutalism" design style.
- ``03-moodboard-web-dark-warm``: Web-based moodboard for a dark and warm design aesthetic.
- ``04-moodboard-web-neo-brutalism``: Web-based moodboard exploring Neo-Brutalism elements.
- ``05-demo-mobile-vanillaJS``: Early mobile app demo built with Vanilla JavaScript.
- ``06-corporate-design``: Corporate identity assets including logos, icons, and font definitions.
- ``07-presentations``: Project presentations covering the product and the Scrum workflow.
- ``08-gantt-diagram``: Project planning resources including a Gantt diagram and related documentation.
